Introducing Quick_J

18-05-2026
By Sofia Maldonado

Today I am introducing Quick_J, a simple tool for (very) early Java devs like myself.

The premise is very simple: this is a shell script that automates the two commands needed to both compile and run a .java program. It works like this:
quick_j Foo.java
This command runs javac Foo.java and java Foo together, so that execution is quicker for the user. This is a useful learning tool, as you will probably have to re-run the same script multiple times in a short timespan for small changes.

This program is part of sshuts, a collection of simple CLI tools I've been making, mostly in pure sh. Be sure to check them out, I actually use them a lot.

Quick_J was inspired by my own Quick_C, which I created earlier this year to do the same but with C programs. You can check out an overview of that script here.

To use the script, simply download the quick_j.sh file, remove the .sh extension and add it to your bin folder, giving it the necessary permissions to run on your computer.

That's it. If you happen to use the program and find any issues with it, please contact me about it in the GitHub repo itself or via email.
