Chofas Archive - About

Welcome to my website! I created this site mostly to learn the basics of HTML, CSS and JavaScript. This is the second iteration of the site, with improved accessibility for mobile devices. The tooling is very barebones, only utilizing one external JS library to work with CSVs.

The design of the homepage was inspired by the look of print newspapers, particularly The New York Times which I've read daily since I was 16. (fun fact: I've collected some specific covers from trips I've made to the US, I'll post pictures below soon).

If you're interested in learning more about me specifically, you can click here. There are also some easter eggs in the page's source code, so get looking!
